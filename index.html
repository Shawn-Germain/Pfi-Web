Destiny2

La nature de ce site est de présenter le jeu qui, en dehors de l'école, est l'activitée sur laquelle je perds le plus de mon temps. Les pages, en ordre, présentera les trois prochains sujet à propos du jeu :

  • Les saisons
  • Le gameplay
  • Le loot
